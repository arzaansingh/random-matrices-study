\subsection{Reproducibility Conventions}\label{app:reproducibility}

Every figure in this report is generated by one of six Python scripts in the \texttt{notebooks/} directory of the accompanying repository. The scripts share the following conventions.

\begin{itemize}
  \item \emph{Libraries.} Python~3 with \texttt{numpy}, \texttt{scipy}, and \texttt{matplotlib}. The MNIST experiments of Section~\ref{subsec:mnist} additionally use \texttt{scikit-learn}, both to download the data set and for the \(k\)-means step.
  \item \emph{Randomness.} Each script fixes a single master seed at the top of the file and draws all randomness from \texttt{numpy} generators of the form \texttt{np.random.default\_rng(SEED + k)}. In scripts \texttt{04} and \texttt{05} each figure has its own offset \(k\); scripts \texttt{00} through \texttt{03} mostly reuse the master seed itself, with offsets only where an experiment needs a separate stream. Either way, re-running a script reproduces its figures exactly. The master seeds are \(2026\) for scripts \texttt{00} through \texttt{03}, \(4040\) for script \texttt{04}, and \(5050\) for script \texttt{05}.
  \item \emph{Eigendecompositions.} Symmetric matrices are diagonalized with \texttt{np.linalg.eigh} or \texttt{np.linalg.eigvalsh}, which return eigenvalues in ascending order; the top eigenpair is read off the last entries. The large MNIST Gram matrices are handled differently; see Section~\ref{app:numerical-notes}.
  \item \emph{Styling.} A shared module, \texttt{notebooks/thesis\_style.py}, fixes the color palette, fonts, and figure sizing, so that every figure in the report speaks the same visual language.
  \item \emph{Parameters.} The parameters of each experiment (the dimension \(p\), sample size \(n\), aspect ratio \(y\), number of replications \(R\), and the spike or signal strengths) are stated in the caption of the corresponding figure and are not repeated here.
\end{itemize}

\subsection{Numerical Notes}\label{app:numerical-notes}

\begin{itemize}
  \item \emph{Exact standardization constants.} Wherever simulated eigenvalues are standardized for comparison with a limit law, the theoretical constants are used rather than empirical ones: the Tracy-Widom comparisons of Sections~\ref{subsec:scm-highdim} and~\ref{subsec:fluctuations} use Johnstone's finite-sample centering and scaling \(\mu_{np}, \sigma_{np}\), and the spike fluctuation comparisons of Section~\ref{subsec:fluctuations} use the exact asymptotic variance \(\sigma_\alpha^2 = 2\alpha^2 \Psi'(\alpha)\) of Theorem~\ref{thm:fund-fluct}. Agreement between histogram and curve is therefore a genuine test of the theory, with no fitted parameters. (The one exception is the Kolmogorov-Smirnov test of Section~\ref{subsec:scm-highdim}, which additionally standardizes by sample mean and standard deviation, as noted there.)
  \item \emph{Large Gram matrices.} For the ten-class MNIST experiment, \(n = 54{,}000\), so the Gram matrix \(\mat{G} = \frac{1}{p}\mat{X}\transpose\mat{X}\) would have about \(2.9\) billion entries and was never formed. Instead, the top right singular vectors of \(\mat{X}/\sqrt{p}\), which are exactly the top eigenvectors of \(\mat{G}\), are computed iteratively with \texttt{scipy.sparse.linalg.svds}.
  \item \emph{MNIST processing.} The \(70{,}000\) images are downloaded once (\texttt{mnist\_784}, version 1, via OpenML), scaled to \([0, 1]\), and sorted by label; each experiment then draws its images per digit without replacement using the script's seeded generator. Following the published code of \citet{LebeauChatelainCouillet2024}, the top eigenvector is centered and renormalized before reading off signs. True labels are used only for evaluation: to estimate the alignment \(\widehat{\zeta}\), to fix the sign convention, and, in the ten-class experiment, to match cluster labels to digits for scoring (by the assignment maximizing agreement, computed with \texttt{scipy.optimize.linear\_sum\_assignment}).
\end{itemize}
